### Date
December 30, 2025

### New Features and Improvements

* XPass Q2 (XPQ2-DB, XPQ2-APB) is supported.
* Supports the [Master Admin]({{'/api/masteradmin' | relative_url}}#MasterAdmin) setting, which grants full administrator privileges.
* Supports the [Device License]({{'/api/devicelicense/' | relative_url}}#DeviceLicense) that enables additional features.
* [QR Config]({{'/api/card/' | relative_url}}#QRConfig) allows you to enable QR code scanning using your device's visual camera. To use this feature, you must first set a license activation key to your device using [Device License]({{'/api/devicelicense/' | relative_url}}#DeviceLicense).
* Supports the [Lock Override]({{'/api/card/' | relative_url}}#LockOverride) feature, which allows you to enroll a card that can open a door to be opened using a specific card.
* Supports the [Door Mode Override]({{'/api/door/' | relative_url}}#lock-unlock) feature, which restores the door to its normal state after the administrator changes the door status and a certain period of time has elapsed.
* Supports the [Realtime device I/O states]({{'/api/event' | relative_url}}#DeviceIOStates) feature to check the current status of device Input/Output ports.
* Supports the [User Override]({{'/api/user' | relative_url}}#UserOverride) feature, which allows to set the door to remain open longer for specific users.
