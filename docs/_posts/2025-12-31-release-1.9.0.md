### Date
December 30, 2025

### New Features and Improvements

* XPass Q2 (XPQ2) is supported.
* [MasterAdmin]({{'/api/masteradmin' | relative_url}}#MasterAdmin) is supported
* [DeviceLicense]({{'/api/devicelicense/' | relative_url}}#DeviceLicense) is supported
* [QRConfig]({{'/api/card/' | relative_url}}#QRConfig) allows you to enable QR code scanning using your device's visual camera. To use this feature, you must first set a license activation key to your device using [DeviceLicense]({{'/api/devicelicense/' | relative_url}}#DeviceLicense).
* [LockOverride]({{'/api/card/' | relative_url}}#LockOverride) is supported.
* [DoorModeOverride]({{'/api/door/' | relative_url}}#lock-unlock) is supported.
* [Realtime device I/O states]({{'/api/event' | relative_url}}#DeviceIOStates) is supported.
* [UserOverride]({{'/api/user' | relative_url}}#UserOverride) is supported.
